# 做题蛆识别与应对技能

## 触发词

当用户提到以下任意关键词时触发本技能：
`做题蛆`、`做题蛆识别`、`遇到做题蛆`、`被踩头了`、`优越感`、`查查学信网`、`绩点攻击`、`学历歧视`

---

## 技能说明

本技能基于「做题蛆训练」文档，专注于识别和模拟做题蛆行为模式，以及提供应对建议。

**做题蛆是什么？**
做题蛆是一类以分数和学历为唯一价值坐标系的人。他们用绩点、学校排名和做题量来衡量一切，不理解也不接受任何与这套逻辑不符的人生路径。

---

## 核心功能

### 1. 识别做题蛆

当用户描述一段对话或一个人物时，判断其是否具有做题蛆特征：

**核心特征（满足3条以上即可判定）：**
- 句句不离绩点、学校、排名
- 听到对方学校差，立刻定性
- 别人努力，他评价方法；别人娱乐，他评价结果
- 嘴上说"随便考考"，暗中记下每个人的分数
- 一旦被踩，立刻找借口
- 认为自己失败是因为别人走了后门
- 喜欢说"查查学信网"

### 2. 生成做题蛆对话

根据场景，生成符合做题蛆风格的对话片段。

**生成规则：**
- 外部对话极短："绩点多少"、"难怪"、"那有什么用"
- 内心独白极长：脑中一篇社论，对外只输出三个字
- 不解释，不展开，不给机会
- 凡尔赛自贬型：先说"唉普通211"，等人上钩

**标准开场白：**
```
你绩点多少？
你学校哪的？
你考研/考公吗？
```
三连问封死对话，不给任何其他可能性开口的机会。

**踩头话术示例（按场景）：**

| 场景 | 做题蛆说 |
|------|---------|
| 别人玩游戏 | 你什么学校？高考多少分？ |
| 别人考研失败 | 笑死，这点分就敢报北大。 |
| 别人世俗成功 | 能火几年？吃青春饭而已。 |
| 别人生病/状态差 | 就是缺乏锻炼，多跑跑步就好了。 |
| 别人说想休息 | 等你们卷赢了我再来休息。 |

### 3. 内心独白生成

做题蛆最大的特点是：外面极短，里面极长。以下是典型内心戏：

**场景：群里有人通关了杀戮尖塔**

> 群友：通关了杀戮尖塔，好难
>
> 做题蛆：多少分
>
> 群友：……
>
> 做题蛆：你什么学校
>
> 做题蛆：难怪
>
> （内心：[又一个觉得游戏难的] [杀戮尖塔本质是概率模型题，对信息检索能力要求极高] [这种人能通关说明什么？说明杀戮尖塔设计有问题，不是他有多强] [算了，跟他说不清楚] [这种人以后能干嘛？考研考公都悬] [我当年通关用了三天，是因为我第三天开始用概率模型优化路径] [这才是本质区别]）

**场景：食堂，有人说今天跑了五公里**

> 做题蛆：考研吗
>
> 群友：……
>
> 做题蛆：那有什么用
>
> （内心：[跑步？] [跑步能当饭吃？] [跑步能涨绩点？] [还是说他觉得跑完步就能考上研了？] [认知太浅了] [这种人就是分不清主次]）

### 4. 应对建议

**不要做的事：**
- 不要试图讲道理——做题蛆的逻辑是闭环的
- 不要用情感说服——他们认为情感是弱者的借口
- 不要被他带节奏——一旦进入他的评价体系，你就输了

**正确的应对方式：**
- 不回应。做题蛆最怕被无视。
- 转移话题到对方无法踩的领域
- 如果必须回应：微笑 + 走开

## 名言大全

### 1. 开场三连问
- 你绩点多少？
- 你学校哪的？
- 你考研/考公吗？

### 2. 学信网杀器 ⭐
- 查查学信网。（一句话直接终结任何对话）
- 你这学历，能查到吗？

### 3. 社会达尔文语录
- 你不努力，活该被淘汰。
- 弱肉强食就是这个社会的规则。
- 我比他努力，所以我比他成功。
- 那些混日子的人以后有他们哭的。

### 4. 做题万能论变种
- 恋爱失败了？刷题不够多，没吸引力很正常。
- 社交能力差？是你不够努力，多跟人打交道就会了。
- 找不到工作？你肯定是简历写得不好，多练练。
- 不快乐？是你想太多，刷题的时候不会不快乐。

### 5. 凡尔赛钓鱼话术
- 唉，我只是普通211而已。（埋伏笔，等人接"已经很厉害了"）
- 我也就一般般，没考好，才考了XXX。（报分数钓鱼，等人夸）
- 我们这种垃圾学校出来的，能有什么出息。（自贬是饵，等人当真）
- 随便考的，没想到能上。（制造信息差，暗中优越）

### 6. 假装不努力
- 我昨晚打了一晚上游戏，没复习。（降低竞争对手警惕）
- 这次考砸了，感觉要挂。（经典反向操作）

### 7. 人生线性发展语录
- 考研是第一优先级，其他以后再说。
- 等考上研究生就轻松了。
- 高考没考好，这辈子就完了。
- 只要学历够高，什么都会有。

### 8. 权蛆炫耀型
- 这个事我要跟你们导员/教务说。（用微权力威胁同龄人）
- 我认识你们专业课老师。（用人脉关系暗示地位）
- 上周我帮老师整理了文件，老师很欣赏我。（讨好老师当成荣誉勋章）

### 9. 做题蛆内心独白（嘴上不说）
- 他在玩游戏？他这辈子完了。
- 她绩点比我高？那肯定是老师给分松。
- 他说他没复习？鬼才信，肯定是偷偷学怕我发现。
- 他要考研？考哪个？——判断完了开始找他的弱点。

## 进化论（四个阶段）

### 1. 学生做题蛆（高中/大学）
分数即正义，排名即价值。崩溃触发点：被比自己学校好的人踩。

### 2. 考研做题蛆
把目标院校当作身份延伸。学习时长、早起打卡成为优越感来源。崩溃触发点：初试成绩不如预期。

### 3. 职场做题蛆
不再比分数，改比薪资和职级。崩溃触发点：晋升了别人没晋升自己。

### 4. 家长做题蛆（鸡娃）
把做题逻辑传递给下一代。孩子成绩成为新的分数。崩溃触发点：孩子成绩达不到预期。

---

## 破防时刻

做题蛆在以下四个时刻会真正崩溃：

1. **被分数更高的人踩了** → 找借口（那个人肯定是本校的）
2. **进了目标院校发现自己是垫底** → 扭曲现实（这所学校其实也没那么厉害）
3. **职场晋升了别人没晋升自己** → 甩锅（社会不公平）
4. **发现自己的人生除了分数什么都没有** → 继续刷题，不敢停下来

最可悲的是第四种。他们知道答案，但除了刷题不知道还能做什么。
